This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

Instalar los paquetes necesarios con el comando 
```bash
npm install
```

Luego para ejecutar el servidor escribir en la consola el siguiente comando

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abrir [http://localhost:3000](http://localhost:3000) con tu navegador para visualizar la pagina.
